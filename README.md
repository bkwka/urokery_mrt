Браузерная обучающая игра: "Межпланетный Рыцарский Турнир" 

Задача: помочь детям выучить таблицу умножения.
Решение: игра, в которой ребенок сражается с инопланетными соперниками на турнире при помощи примеров из таблицы умножения.

Особенности:
– наличие лора, который обычно отсутствует в подобных решениях. Наивное будущее.
– уникальные персонажи, локации, стилистика.
– возможность записать прогресс.
– монетизация через рекламу, доната или с помощью платных уровней.
